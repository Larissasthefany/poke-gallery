# 🧩 PokeGallery

Projeto front-end que consome uma API de Pokémon e exibe os dados em formato de cards interativos.

Este é meu **primeiro projeto utilizando consumo de API**, com foco em prática de JavaScript moderno, manipulação do DOM e organização de código.

---

## 🚀 Funcionalidades

* 🔎 Busca em tempo real por nome do Pokémon
* 🔤 Filtro ignorando maiúsculas/minúsculas
* 📄 Paginação (próximo / anterior)
* 🧠 Renderização dinâmica de cards
* ⚠️ Mensagem quando nenhum resultado é encontrado

## 📡 Consumo de API

O projeto utiliza a API pública de Pokémon:

* PokéAPI (REST)

Os dados são buscados dinamicamente e transformados em elementos visuais na tela.
